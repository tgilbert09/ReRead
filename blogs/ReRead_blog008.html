Prototype done!

25th September 2024

Two nights ago, on 23rd September, ReRead launched "The Library” of charity shop books!! 🎉🎉🎉 Here’s what it looks like!

While developing the back end, I used the same photo over and over again. I can’t tell you how many times I’ve seen “Gerald Brenan, the Interior Castle” on my screen. For some reason, Mask R-CNN (the machine learning model I am using to detect and cut out books) always picked this book first, and now it’s the first book on the website too!

There are many features still missing from the website, like a working search bar, but don't worry - these will be coming soon :)

It's been a while since my last blog post, but during that time, I’ve learnt so much. I've been diving into Python, Javascript, Cloudflare (R1, D1, Images), SQL databases, machine learning (PyTorch, Segment Anything 2, Mask R-CNN, text recognition) and even dabbling in a bit of Swift! I’m having an amazing time and don’t want this to stop.

Next up.. I need to figure out a revenue model for ReRead so I can keep this going long-term.
